Get current IP address
hostname -I

Configure settings
sudo raspi-config

Get list of wi-fi networks available
sudo iwlist wlan0 scan

Shutdown the system
sudo shutdown -h now

Test a LED light from console
gpio -g mode 21 out (21 is pin nr.)
gpio -g write 21 1 (pin 21 power on)
gpio -g write 21 0 (pin 21 power off)

Update System
$ sudo apt-get update
$ sudo apt-get dist-upgrade

$ /etc $ sudo nano rc.local

Start new VNC machine in 4K
$ vncserver -geometry 4096x2160

Set console to BASH
source ~/.bashrc
